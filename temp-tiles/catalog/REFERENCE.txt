  objects-misc: rows 12-35 — Cacti, rocks, snow, hedges (rows 12-35)
  objects-large: rows 36-64 — Giant trees, vines, logs (rows 36-64)
  tiles-terrain: rows 65-98 — Grass, dirt, sand, stone, paths (rows 65-98)
  tiles-water: rows 99-130 — Brick, decorative, water (rows 99-130)

COORDINATE SYSTEM:
  Row 0 = top of tileset
  Col 0 = left of tileset
  Each tile = 32×32px
  In catalog images: 2× scale, red grid lines between tiles

HOW TO USE:
  1. Open section images in an image viewer
  2. Count tiles from top-left (row, col) using red grid
  3. Add coordinates to extract-objects.cjs or extract-ground-tiles.cjs
